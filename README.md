
# Challeng CPP Intermediante 2026

Continuação direta do repositório de desafios iniciantes. Aqui o foco muda: não é mais só usar as ferramentas do C++, é entender como elas funcionam por dentro e construir as próprias quando precisar.

Nos 5 projetos iniciantes aprendi a base — loops, funções, classes, arquivos, STL. Agora o objetivo é ir mais fundo: herança, memória, genéricos, erros estruturados e estruturas de dados do zero.

Cada projeto tem o código completo, um roadmap documentando o que foi aprendido e os bugs reais que apareceram no caminho.

## Os 5 desafios

1. **Sistema de Combate com Herança** — polimorfismo, classes base e filhas, métodos virtuais
2. **Gerenciador de Memória com Ponteiros Inteligentes** — heap, unique_ptr, shared_ptr, propriedade de objetos
3. **Container Genérico com Templates** — escrever código que funciona com qualquer tipo
4. **Sistema de Exceções e Tratamento de Erros** — throw, catch, classes de exceção próprias
5. **Lista Encadeada do Zero** — estrutura de dados sem STL, manipulação manual de ponteiros
